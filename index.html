The Basic Elements

The Hypertext Markup Language, HTML, uses a series of tags to define different sections of your HTML page. Some basic tags include:

  • <h1>
  • <p>
  • <div>
  • <img>
  • <hr>

    • For this exmaple lets look at the use of the h1 tag.h1 is usually the title of your page, or the header. It its default is large font. An h1 tag, like most tags in HTML, has an opening and closing tag, like this:
    <h1> </h1>

    The h1 element tells the browser the main heading of your webpage. In addition, the tags h2, h3, h,4, h5 & h6 indicate subheadings in descending order of prominence. The div tag is also a very important tag for starting HTML. It is used to define a devision or section, and can be used as a container for many HTML elements to style with JavaScript or CSS

    A few other important tags to begin using in HTML are p tags for paragraphs, img tags to insert images, and hr tags for a 'horizontal ruler' to indicate a seperator between sections. It is important to note that an <img> tag is a single tag, and does not have closing tag. The <img> tag contains a source, and can specify height or width by pixels:
    <img src="picture_file.png" height="150" width="150">
    We will discuss the img tag in greater detail in the Images and Anchor Links section of this webpage.

HTML 5 Elements

HTML 5 elements include main, header, footer, nav, video, article, section, and others. These elements help Search Engine Optimization and are more descriptive. Specifically, the main tag indicates the main content of your page, for search engines or other developers reading over your code.Header and footer elements define exactly that: the header to your page and the footer to your page.Article and section elements are used for content within your page. The article tag is used for "independent self-contained content", and can be nested within a section tag A video tag us used to embed a video. A very common and important element in HTML 5 is the <nav> element. It defines a set of navigational links, such as in a navigation bar just like the one on the far left side of this website. Here is an example of a nav element with clickable links inside of it:
<nav>
<a href="/html/">HTML</a> |
<a href="/css/">CSS</a> |
<a href="/js/">JavaScript</a> |
<a href="/jquery/">jQuery</a>
</nav>

As you can see, the nav element uses <a href=""> to link to different elements that appear on the webpage. When these links are clicked the user is brought to the corresponding section of the webpage designated by the href location.

Images and Anchor Links

To add images to your web page, use the <img> tag and a source. The image tag is self closing, so it does not need an end or closing tag. The src is the source from where the image is retrieved, be it a https link or a file from your computer such as a .png file. Within the image tag should be an alt attribute. This will provide a text alternative if the browser can not display the image. It is also used by screen readers for the visually impared to know what images are being displayed on the web page. An example of using the image tag with the alt attribute can be seen as follows:
<img src="https://www.image.com/image.jpg" alt="A picture of some HTML text.">

The anchor element links to either another web page. It containts an href attribute, which will be the link to the web page, and some text that will be displayed as a link. An example of the anchor element with a href attribute can be seen as follows:
<a href="https://www.SomeWebPage.com">This will bring you to SomeWebPage.com</a>
The underlined hyperlink text indicates a clickable element that will take you to a new website.

Ordered and Unordered Lists

HTML and HTML5 use list elements to create both ordred and unordered lists. This can be useful for web pages that offer ordered directions such as how to change a tire, or unordered lists for things like ingredients for a recipe. Within these lists are list items, represented as li.An unordered list might be written like this:
<ul>
<li>Butter</li>
<li>Milk</li>
<li&gEggs</li>
<li>Flour</li>
</ul>
 and look like this:

  • Butter
  • Milk
  • Eggs
  • Flour

An ordred list may be useful in writing directions that need to be followed step by step. For example, an ordred list desribing how to change a tire could be written as:
<ol>
<li>Palce the jack under the car</li >
<li>Prop up the car with the jack, exposing the tire you want to change</li>
<li>Use a tire iron to loosen the bolts on the tire</li>
<li>Take off the old tire and put on the spare</li>
<li>Tighten the bolts on the spare, securing it to the car</li>
</ol>
and look like this:
  1. Place the jack under the car
  2. Prop the car wit the jack, exposing the tire you want to change
  3. Take off the old tire and put on the spare
  4. Tighten the bolts on the spare, securing it to the car

If you want to submit data to a server you can use the form element to do so! You can use the action attribute to submit the data to a specific URL, like such:

The div Element

The div or "division" element can be used as a container for multiple elements, and is likely the most common HTML element. The div element is not self closing, so it needs an opening and closing tag, <div></div>. It us used for a division or a section of a web page and can be used to add CSS style elements.

One example of the many things you can do with the div element is to wrap lists in them. For example, a list wrapped in a div element could look like this:
<div class="animalList">
<ul>
<li>Dogs</li>
<li>Cats</li>
<li>Birds</li>
</ul>
</div>
 By using the div tag or referencing the class attribute of animalList, you can style the entire div tag with CSS. Div containers are great to use if a large portion of your HTML page need to be styled similarly and you do not want to style each individual element.
